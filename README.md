Hotel Booking Analysis (SQL \& Power BI)



\##Project Overview

This project explores hotel booking data to identify key revenue drivers and booking behaviors. I performed Exploratory Data Analysis (EDA) using SQL Server and developed an interactive dashboard in Power BI to visualize performance metrics.



\##Key Insights (from Dashboard)

•Total Revenue:\*\* $14.79M

•Average Daily Rate (ADR):\*\* $104.44

•Average Discount:\*\* 25.80%

•Total Nights Booked:\*\* 367.94K



\##Tools Used

\* \*\*SQL Server (SSMS):\*\* Used for data cleaning, handling null values, and initial EDA.

\* \*\*Power BI:\*\* Used for data modeling, DAX calculations (like Average Discount %), and visualization.



\## How to Use this Repo

\*SQL\_Queries.sql\*\*: Contains the code used to extract and clean the data.

\*Hotel\_Analysis.pbix\*\*: The Power BI file containing the interactive dashboard.
